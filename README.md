# Academic Predictor for AUI

An AI-powered application that helps students at American University in the Emirates (AUI) predict and understand their academic performance.

## What It Does

The Academic Predictor is an intelligent tool designed to:
- **Analyze academic patterns** - Uses Google's Gemini AI to process and analyze student performance data
- **Provide predictions** - Offers data-driven predictions about future academic outcomes
- **Support decision-making** - Helps students make informed decisions about their courses and study strategies
- **Personalized insights** - Delivers tailored recommendations based on individual academic profiles

## How It Works

This app leverages Google's Gemini AI Studio to create an interactive, conversational interface where students can:
1. Input their academic information
2. Ask questions about their academic trajectory
3. Receive AI-powered analysis and suggestions

## Getting Started

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
